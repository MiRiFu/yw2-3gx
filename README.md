## ダウンロードの仕方について
[リリース](https://github.com/MiRiFu/yw2-3gx/releases)からバージョンを選び下にある<ins>**Assets**</ins>をクリックしYW2_CTRPF_XXXでかつ拡張子がzipであるものをダウンロードするだけです。
